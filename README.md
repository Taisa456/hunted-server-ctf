# The Haunted Server

A few weeks ago I decided to build my first CTF room from scratch.
No tutorials, no templates - just a concept, a Linux VM, and a lot
of trial and error.

The result is this: a beginner-friendly room with a story behind it.
A server went dark. An admin disappeared. 
Something is hiding in the noise.

## What to expect

The room has 4 stages that chain into each other naturally.
No stage can be skipped - each answer leads to the next question.

You'll need to think like an investigator, not just a hacker.

**Difficulty:** Beginner - but not everything is obvious at first glance.

---

## Tools you might need

Do your research. Part of the challenge is figuring out
which tools to use and when.

A basic Linux environment is all you need to get started.

---

## How it was built

Free Oracle Cloud Ubuntu VM.
Self-hosted CTFd scoreboard.
Total build time: about 2 days.

Learned a lot along the way - especially about how small
misconfigurations can lead to full system compromise.
